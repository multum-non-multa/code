## Math Operations

Python has 7 arithmetic operators:

`+` for addition

```python
>>> 1 + 2
3
```

`-` for subtraction

```python
>>> 1 - 2
-1
```

`*` for multiplication

```python
>>> 1 * 2
2
```

`/` for division

```python
>>> 1 / 2
0.5
```

`//` for floor division

```python
>> 1 // 2
0
```

`**` for exponentation

```python
>>> 3**2
9
```

`%` => modulus

```python
>>> 4 % 3
1
```

### Uses of `%`

- find even numbers

- divisible by x
